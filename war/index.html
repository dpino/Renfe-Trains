Query Renfe Trains

This is a small experiment I coded basically for playing and learning about Google Application Engine. It was inspired by Trenes, an iphone app built by an old friend of mine :) This web app lets you query the Spanish Railway System (Renfe) time schedule, searching timetables by origin-destination and date. Results are queried via Renfe website. This service comes with NO WARRANTY. Please, use Renfe website instead.

Trains from Vigo to Ourense leaving today (output XML)
http://pinowsky.appspot.com/trains?origin=vigo&destination=ourense
Trains from Vigo to A Coruña leaving tomorrow (output XML)
http://pinowsky.appspot.com/trains?origin=vigo&destination=a coruña&date=tomorrow
Trains from Vigo to Madrid today (output JSON)
http://pinowsky.appspot.com/trains?origin=vigo&destination=madrid&output=json
Trains from Vigo to Barcelona leaving on 1st of September 2010 (output JSON)
http://pinowsky.appspot.com/trains?origin=vigo&destination=barcelona&date=2010-09-01&output=json
